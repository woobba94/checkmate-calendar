# checkmate-calendar
### 연동과 일정 공유가 가능한 캘린더 서비스

---

## 서비스 특징

- **누구나 사용하기 쉬운 UI/UX**
- **구글 캘린더 연동 지원** (타 캘린더 추가 연동 지원 예정)
- **유저 간 캘린더 공유 가능**
- **복수 캘린더 선택/병합 보기**
- **모바일 앱** (지원 예정)
- **다양한 위젯** (지원 예정)

## 기술 스택

- **Frontend**
  - React
  - TypeScript
  - Vite
  - React Query
  - Sass/SCSS
- **Backend/Infra**
  - Supabase (인증, DB, Edge Functions, Google Calendar 연동)
  - Vercel (FE 배포)

## 시작

```bash
npm i
npm run dev
```

## 배포

- Vercel 자동 배포
